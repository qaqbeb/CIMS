6.将收银台的订单显示改为只显示零售的，已审核的订单
7.给收银台添加两个按钮，分别是开单和退货，在开单过程中，下了单之后等待客户付款，待客户付完款后进行审核，审核完订单的状态直接变为已审核状态。如果有客户想要退货，可以点击退货按钮，会同时销毁订单以及将货品归还回仓库
